O foco principal deste módulo foi consolidar a base de algoritmos, entender a sintaxe da linguagem e aprender a manipular dados de forma eficiente.

---

## 🎯 Objetivos Concluídos

Nesta primeira etapa, foram desenvolvidos **35 exercícios práticos** cobrindo os seguintes pilares da engenharia de software:

* **Configuração do Ambiente:** Instalação do ecossistema Python, uso do IDLE e integração com IDEs de mercado.
* **Tratamento de Dados e Cálculos Matemáticos:** Manipulação de tipos primitivos (`int`, `float`, `bool`, `str`), operadores aritméticos e ordem de precedência.
* **Módulos e Bibliotecas:** Importação e utilização de pacotes nativos e externos (como `math`, `random` e `pygame`).
* **Manipulação de Texto:** Análise, fatiamento (`slicing`), transformação e busca de strings.
* **Estruturas Condicionais:** Tomada de decisões no código utilizando estruturas simples e compostas (`if/else`).
